# MPU6050_F28379d
In this code we have a working MPU6050 readding with TMS320F28379d Lauchpad from Texas Instruments.
I used I2CA module on the GPIO0 and GPIO1.

| Lauchpad Pins  | MPU6050 Pins |
| ------------- | ------------- |
|     3.3V      |    Vcc        |
|      GND      |     GND       |
| J4(40) SDA  | SDA  |
| J4(39) SCL  | SCL  |
